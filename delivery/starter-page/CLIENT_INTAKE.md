# Client intake

Use this before starting a paid starter page.

## Required

- Business/project name:
- Page goal:
- Target audience:
- One action visitors should take:
- Existing website or profile:
- Reference page/style:
- Text/copy:
- Logo/images:
- Contact or booking URL:
- Deadline:

## Scope confirmation

```text
Confirmed scope:

- one page or one focused fix
- mobile-friendly layout
- clear CTA/contact flow
- source files
- one small revision
- first usable version within 24 hours after materials are received
```

## Risks to clarify

- Does this need payment processing?
- Does this need login/account features?
- Does this need custom backend/database?
- Does this need copywriting from scratch?
- Are there brand guidelines?

Anything outside the starter scope should be quoted separately.
